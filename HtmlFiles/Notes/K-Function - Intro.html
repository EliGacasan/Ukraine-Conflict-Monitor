What are r-neighbours?

The number of r-neighbours a point X in a point pattern data has, is the number of points lying with a circle of radius r, centered at the point X.

What is the K-Function?

The K-Function is a very popular technique for analyzing spatial correlation between points. The assumption required before applying the K-Function is that the point process is stationary. The rationale for interpretating the K-Function is by comparing it with the theoretical K-Function of a homogeneous poisson process with the same average intensity. In particular, there is line denoting Kpois(r), which is the theoretical K-function for homogeneous poisson point pattern, and a line that denotes the empirical K-Function calculated from the data (the selected point pattern data). To quote from the book again, the empirical K-Function is the cumulative average number of data points lying within a distance r of a typical data point, corrected for edge effects, and standardised by dividing by the intensity. The standardisation and edge correction make it possible to compare point patterns with different numbers of points, observed in different windows. If the empirical K-Function calculated from the data is above the theoretical K-Function for a homogeneous poisson process with the same average intensity, this indicates that the point pattern of interest has more r-neighbours than would be expected for a completely random pattern (that of a homogeneous poisson process in which we are comparing), indicating clustering. If the empirical K-Function calculated from the data is below the theoretical version, this indicates that the point pattern of interest has less r-neighbours than would be expected for a completely random pattern, indicating repulsion/regularity/inhibition.

What happens when you interpret the K-Function, assuming a non-stationary process?

If the point process is assumed to be non-stationary, variations between the theoretical and empirical K-Functions may not be evidence of interpoint interaction, because it variations may be attributable to variations in spatial intensity.

Inhomogeneous K-Function

As the name suggest, this is like the K-Function, except that it deals with inhomogeneous point patterns. The assumption behind the inhomogeneous K-Function is that the process is correlation-stationary.
