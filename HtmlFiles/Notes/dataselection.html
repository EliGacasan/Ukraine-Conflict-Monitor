Data Selection and Options Bars

Options bars are located at the top-right of select boxes. The 'Data Visualization' and 'Point Pattern: Introduction and Data Selection' pages are the only pages which contain boxes that allow you to filter data. Options boxes in the 'Point Pattern' pages allow you to change different aspects of the graphics.

Data Visualization

The options bar in the 'Time Series Visualization' box will change the data used for both the 'Time Series Visualization' and 'Multiple Bar Chart' boxes. The options bar in the 'Maps' box will change the data used for both the 'Event Distributions' and 'Maps' boxes.

Point Pattern: Introduction and Data Selection

Click on any number of the administrative regions (tiles) in the 'Selection Map' box or click on the top-right options button to specify the data to be used in the 'Point Pattern:Intensity', 'Point Pattern:Correlation', and 'Point Pattern:Spacing' pages. After clicking on a tile, some data should appear in the 'Point Pattern' box. Click on the options button of the 'Point Pattern' box in order to further filter the data according to event type and date. The filtered data will specify a object of class ppp (2-D point pattern) in R and the rotate option will simply rotate the specified pattern. The data selected in these two tabs will be used for all exploratory analysis boxes in the 'Point Pattern' pages.
