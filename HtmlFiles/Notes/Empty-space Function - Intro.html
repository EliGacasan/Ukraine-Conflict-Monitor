What is the empty space function?

Mathematically in a 2-D plane, the empty space distance can be defined as d(u,X) = min{||u − xi|| : xi ∈ X}, where u is a arbitrary fixed location, X is a point pattern, and xi is a point in the point pattern. The empty-space function, defined for all distances r ≥ 0, is F(r) = P{d(u,X) ≤ r}. This is the cumulative distribution function of the empty-space distance and may be interpreted as follows: The values of F(r) are probabilities (between 0 and 1) giving, for any fixed reference location u, the chance that there will be a point of X lying within distance r of this location.. In a similar fashion to the K-Function, we compare the empirical empty space function estimated from the data, with the empty space function for a homogeneous poisson process (for a completely random pattern). The estimator used in this dashboard for the empty space function assume that the point process is stationary, and experience suggests that the choice of edge correction is not very important so long as some edge correction is performed.

Interpretation

The interpretation of the empirical empty space function F(r), is opposite of that of the K-Function, if the empirical empty-space function is above the theoretical version for a homogeneous poisson process with the same average intensity, this means for for a given distance r, the estimated probability that d(u,X) ≤ r is greater than it would be in a completely random pattern. Consequently the empty-space distances are shorter than expected if the pattern were completely random. This is consistent with a regular pattern. Similarly if the empirical empty space function estimated from the data is below the theoretical version for a homogeneous poisson process with the same average intensity, this means that the probability for a given distance r, that a arbitrary location u will have a point of X lying within a distance r is lower than that of a completely random pattern, indicating that the empty-space distances are larger than would be expected if the corresponding pattern were completely random; this is consistent with clustering.

Inhomogeneous Version

There is a inhomogeneous version, details of which can be found in M.N.M. van Lieshout. A J-function for inhomogeneous point patterns. The required assumption is that the 'k-point correlation functions' for all k ≥ 2 are invariant under translations, with the additional assumption that the true intensity function is positive for all locations u.
