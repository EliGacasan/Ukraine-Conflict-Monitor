Navigation and Features

  • Top Header options bar: (left to right)
    1. Title: Capstone Project
    2. Open/Close left sidebar button
    3. Dark/light mode button
    4. Open/Close Information Bar button:
        The information bar contains:
      • Select sections of the ACLED codebook which reviews the data in detail (event definitions, data collection procedures, etc.)
      • Data tables so the user can see what data is being shown via the graphics boxes throughout the dashboard.
  • Left Sidebar: Navigation of main content pages
  • Graphics boxes:
    • Data Visualization
      • Multiple Bar Chart
      • Time Series Visualization (options bar: filter data to be displayed in both the multiple bar chart and time series visualization boxes)
      • Event Distributions
      • Maps (options bar: filter data to be displayed in both the event distributions and maps boxes.
    • Point Pattern: Introduction and Data Selection
      • Selection Map (options bar: specify the administrative regions to be considered in the 'Point Pattern' pages)
      • Point Pattern (options bar: specify the event types and rotation to be considered in the 'Point Pattern' pages)
    • Point Pattern: Intensity
      • Quadrat Counts
      • Kernel Smoothed Intensity
    • Point Pattern: Correlation
      • Fry Plot
      • Function Estimates
    • Point Pattern: Spacing
      • Stienen Diagram
      • Dirichlet Tesselation/Voronoi Diagram
      • Nearest Neighbour Distance Distribution Function
      • Empty-Space Function
